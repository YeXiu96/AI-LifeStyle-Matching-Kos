AI Matching Engine

How the algorithm works

A multi-criteria weighted scoring pipeline that produces a 0–100 compatibility score for every user-property pair.

01

Input Collection — Lifestyle Quiz

User answers 12 questions covering sleep schedule, cleanliness level (1–5), social preference, hobbies, noise tolerance, cooking habits, commute mode, and budget range. Responses are normalized into a feature vector.

02

Property Profile Encoding

Each property is encoded with attributes from owner input and AI-extracted tags: noise level (quiet/moderate/lively), tenant community type, facility tags, location category, and verified amenities.

03

Weighted Cosine Similarity Scoring

The AI engine computes cosine similarity between user preference vectors and property attribute vectors. Each dimension carries a different weight:

25% Budget fit
20% Location/commute
15% Sleep schedule
15% Social vibe
10% Cleanliness
10% Noise tolerance
5% Hobbies alignment
04

Collaborative Filtering Boost

Properties that users with similar profiles have booked and rated highly receive a collaborative filtering score boost, improving recommendations over time as platform data grows.

05

Ranked Output & Score Explanation

Final scores are ranked, deduplicated, and served via API with a per-dimension breakdown (e.g., "Budget: 98%, Location: 91%, Sleep: 85%") so users understand why each property was recommended.
